---
title: CLI - logs
---
 
 Usage: 

    $ sqz logs [options]

 Description:

    Output the logs of a deployed function

 Options:

    --function / -f  function name (required)
      value
 
    --stage          environment stage (optional) "dev"
      value
 

 Examples:

    $ sqz logs 
    $ sqz logs --function MyFunction
