---
title: Development - Runtime
---

Squeezer support multiple runtime languages :

| Name | Code |
| -- | -- |
| Node JS | <code>nodejs</code> |
| Python | <code>python</code> |
| Java | <code>java</code> |
| C# | <code>csharp</code> |

`squeezer.yml` 

```yaml
runtime: nodejs
# runtimeVersion: "x.x.x"
```
